FI-042

PANELWRIGHT

A single-file UI designer for the Arduino Giga Display Shield.

A drag and drop layout tool for LVGL panels, built as one self-contained HTML page. Drop widgets onto a one-to-one 800 by 480 canvas, set their properties, then export a ready-to-flash Arduino sketch as a .ino file or a zipped sketch folder. Supports multiple screens with navigation between them, an LVGL default theme and a Tradigital palette, and eighteen widget types including labels, buttons, sliders, switches, checkboxes, bars, arcs, meters, dropdowns, text areas, rollers, tabviews, lists, charts, LEDs, icons, keyboards, and calendars.

Built for the Arduino Giga R1 + Giga Display Shield combo but the generated sketch follows generic LVGL v8 conventions and adapts easily to other display hardware. No build step, no dependencies, no CDN. Just one HTML file you can keep next to your sketches.
